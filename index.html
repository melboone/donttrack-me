Linux Mint Easiest to use

The purpose of Linux Mint is to produce a modern, elegant and comfortable operating system which is both powerful and easy to use.
Started in 2006, Linux Mint is now the 3'rd most widely used home operating system behind Microsoft Windows, Apple Mac OS.

Fedora Linux Leading edge

Fedora is a community powered Linux distribution, owned by Red Hat. Their main objective is to contain free and open-source software as well as being on the leading edge of such technologies.
Fedora life-cycle is relatively short, maintenance support lasting only 13 months. New Fedora versions are released every 6 months. A very good choice for Desktop users but a pretty poor choice for product-development which requires long-term support.

OpenSUSE Friendly

OpenSuse it's sponsored by SUSE and advertises itself as a easy to use, multi-purpose and complete distribution. Aimed towards users and developers working on a desktop or a server, openSUSE project provides a web portal for the community involvement.

Debian Most stable

Debian Linux is a ethical Linux which supports only free software. Founded in 1993, Debian Linux is used as a base platform for more than 25 other Linux distributions like: Ubuntu and Linux Mint.

Gentoo Advanced features

Gentoo Linux is a meta-distribution which advertises itself as a Linux distribution with extreme performance and configurability with "near-unlimited adaptability".
Gentoo package management is designed to be modular, portable, flexible, easy to maintain and optimised for the user's machine. Gentoo users have configurations and sets of installed programs which are unique to themselves.

openBSD Most secure BSD Family

openBSD is probably the safest distribution in the world. The lase security hole found in the default install was back in 2007 which was quickly fixed. In fact their slogan is "Only two remote holes in the default install, in a heck of a long time!" which refers to the first security hole in 2002 and the latter in 2007.
The whole BSD family is pretty secure and anyone of them is recommended to be used by you on your desktop or on your server.
